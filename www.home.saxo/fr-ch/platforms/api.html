      Foire aux questions

      SaxoOpenAPI est une API de type REST sur le protocole HTTP avec support de streaming. SaxoOpenAPI donne accès à:

      • toutes les ressources et fonctionnalités nécessaires à la création d’une plateforme de trading multi-actifs ultra-performante,
      • des données de référence complètes sur tous les instruments traitables, en allant des noms d’instruments jusqu’aux règles de format, fuseaux horaires des bourses et horaires d’ouverture,
      • une vue d’ensemble sur le portefeuille, y comprisles comptes, les soldes, les positions et les ordres en temps réel,
      • une assistance complète pour les prix en continu, les demandes de prix, les placements d’ordres, les confirmations de transactions, etc.
