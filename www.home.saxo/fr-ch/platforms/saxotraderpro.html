Configuration matérielle minimale :
Système d’exploitation : Windows 8 ou 10, 64 bits, Mac OS 10.11+
Processeur : Intel i3 3e génération (ou mobile-class 6e génération), i5 2e génération (ou mobile-class 4e génération) ou équivalent
RAM : 8 Go
Espace disque disponible : 650 Mo
Écran : 1366x768
Connexion Internet : 2 Mbit/s

Configuration matérielle recommandée :
Processeur : Quad-core; Intel i5 3e génération (ou mobile-class 6e génération) ou équivalent
RAM : 8 Go (16 Go pour plus de deux écrans)
Graphiques : Au moins équivalents à Intel HD 530. 256 Mo de mémoire par écran.
Disque dur : Disque SSD
Écran : 22”, 1 920 x 1 080
Connexion Internet : DSL/Câble 10 Mbit/s
