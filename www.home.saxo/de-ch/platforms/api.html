      Häufig gestellte Fragen

      SaxoOpenAPI ist eine REST-ähnliche API auf dem HTTP-Protokoll mit Streaming-Support. SaxoOpenAPI bittet Zugriff auf:

      • Alle Ressourcen und Funktionen, die zum Aufbau einer leistungsstarken anlageklassenübergreifenden Trading-Plattform benötigt werden.
      • Vollständige Referenzdaten für alle handelbaren Instrumente – von Instrument-Namen über Formatierungsregeln, Zeitzonen der Börsen bis hin zu Öffnungszeiten.
      • Vollständige Portfolioübersicht, einschliesslich laufender Konten, Guthaben, Positionen und Order in Echtzeit.
      • Umfassender Support für Streaming-Angebote, Kursanfragen, Order-Platzierung, Handelsbestätigung etc.
      In diesem Leitfaden erfahren Sie mehr.

      Standardmässig haben Sie kostenlosen Zugang zu online handelbaren Streaming-Kursen im Devisenhandel und zu 5.000 Anleihen. Sie haben zudem kostenlosen Zugang zu Chart-Daten im Devisenhandel.
